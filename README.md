# awesone-multi-vector-ANN

The following informations are generated by Claude Opus 4.8.

A curated list of recent multi-vector (late-interaction) ANN research. "Guarantee" indicates whether the paper provides a *provable* approximation/structural guarantee (✅) or is validated empirically only (❌).

| System | Venue | Core idea | Guarantee |
|---|---|---|---|
| **DESSERT** | NeurIPS 2023 | First method designed natively for vector-set search. Uses LSH to compress each vector set into a sketch and estimates the query–doc similarity matrix by comparing hash codes to recall top-k. Fast, but large index (~107GB on MS MARCO). | ✅ Concentration/approximation bounds on its LSH similarity estimator |
| **MUVERA** | NeurIPS 2024 | Reduces multi-vector (Chamfer) similarity to *single-vector* MIPS via asymmetric **Fixed Dimensional Encodings (FDE)**, so off-the-shelf MIPS solvers apply. ~10% recall gain, ~90% latency cut on BEIR vs PLAID. | ✅ Proves FDE gives an ε-approximation of Chamfer; first *provably* faster-than-brute-force algorithm |
| **α-Reachable Graphs** | ICML 2025 (Oral) | Theoretical model for how *non-metric* similarity (e.g. ColBERT's Chamfer distance) behaves on graph indexes; shows graph indexes (DiskANN) support multi-vector search directly. | ✅ Guarantees on graph reachability under non-metric similarity |
| **Stability** | arXiv 2025 | Analyzes *why* modern vector retrieval stays effective despite very high dimensionality, framed around stability of nearest-neighbor structure. Analytical, not a system. | ✅ Theory/analysis paper |
| **GEM** | SIGMOD 2026 | Native graph index built directly over vector *sets* (set-level clustering + metric decoupling + quantization). Up to 16× faster than SOTA with smaller index than MUVERA/DESSERT. | ❌ Empirical |
| **WARP** | SIGIR 2025 | Efficient serving engine that accelerates ColBERTv2/PLAID-style late interaction while preserving quality. | ❌ Empirical |
| **IGP** | SIGIR 2025 | Proximity-graph candidate generation yielding only *hundreds* of candidates (vs tens of thousands in PLAID/DESSERT/EMVB/MUVERA) at high recall. | ❌ Empirical |
| **EMVB** | ECIR 2024 | Speeds up the PLAID late-interaction pipeline with optimized bit vectors, column-wise SIMD reduction, and product quantization. | ❌ Empirical |
| **LEMUR** | arXiv 2026 | *Learned* multi-vector retrieval, replacing hand-designed indexing/scoring with learned components. | ❌ Empirical |
